# Fast Layers | After Effects Automation

---

## EN

## Key Features

### Null Object & Solids
* **Null Object:** Instant null object creation. The **Link** button parents selected layers to the new null.
* **Solid Layer:** Create color fills. Size can be configured in ⚙️ **Settings**. The **Blind** button parents selected layers to the new solid.

### Adjustment & Camera
* **Adjustment:** Create adjustment layers. The **Chain** button parents selected layers to the new adjustment layers.
* **Camera:** Quick camera setup tailored to selected layers.

### Marker Split
A tool for automatic layer slicing based on markers.
* Supports both Composition and Layer markers.
* Use **🔄** to refresh sources and **Trim** to split layers at every marker point.

### Un-Precompose (New!)
* **Extract:** Instantly extract all layers from selected pre-compositions back to the main timeline. 
* Maintains perfect layer order, timing (startTime), and automatically trims layers to the original pre-comp boundaries.

### Keyframe Tools
* **Remap Keys:** Shift the timing of selected keyframes by a specific frame step.

---

## Installation
1. Copy the extension folder to:
   * **Win:** `C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\`
   * **Mac:** `/Library/Application Support/Adobe/CEP/extensions/`
2. In After Effects, go to `Window > Extensions > Fast Layers`.
3. Enable **"Allow Scripts to Write Files and Access Network"** in Preferences.

## RU

## Основные функции

### Null Object & Solids
* **Null Object:** Быстрое создание нулл-объектов. Кнопка **Link** привязывает выбранные слои к новому нуллу.
* **Solid Layer:** Создание цветных слоев. Размеры настраиваются в ⚙️ **Settings**. Кнопка **Blind** привязывает выбранные слои к новому солиду.

### Adjustment & Camera
* **Adjustment:** Создание корректирующих слоев. Кнопка **Chain** привязывает выбранные слои к новым корректирующим слоям.
* **Camera:** Быстрое создание камеры под текущие нужды сцены.

### Marker Split
Инструмент для автоматической нарезки слоев по маркерам.
* Поддерживает маркеры композиции и маркеры на отдельных слоях.
* Кнопка **🔄** обновляет список источников, а **Trim** нарезает слои.

### Un-Precompose (Новое!)
* **Extract:** Мгновенно вытаскивает все слои из выбранных прекомпозиций обратно на основной таймлайн.
* Сохраняет точный порядок слоев, тайминг и автоматически обрезает их по границам удаляемой прекомпозиции.

### Keyframe Tools
* **Remap Keys:** Изменение тайминга выделенных ключей с заданным шагом в кадрах.

## Установка
1. Скопируйте папку расширения в директорию:
   * **Win:** `C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\`
   * **Mac:** `/Library/Application Support/Adobe/CEP/extensions/`
2. В After Effects перейдите в `Window > Extensions > Fast Layers`.
3. Разрешите скриптам запись файлов в меню `Preferences > Scripting & Expressions`.
