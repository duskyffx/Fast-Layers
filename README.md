# Fast Layers | After Effects Automation

---

## EN

## Key Features

### Null Object & Solids
* **Null Object:** Instant null object creation. The **Link** button parents selected layers to the new null.
* **Solid Layer:** Create color fills. Width and height can be configured in ⚙️ **Settings**. If fields are empty, it defaults to comp size or the size of the selected layer. The **Blind** button parents selected layers to the new solid layers.

###  Adjustment & Camera
* **Adjustment:** Create adjustment layers. The **Chain** button parents selected layers to the new adjustment layers.
* **Camera:** Create camera.

### Marker Split (New!)
A tool for automatic layer slicing based on markers.
* Supports both Composition markers and Layer-based markers.
* Use the **🔄** button to refresh the list of available marker sources.
* The **Trim** button splits selected layers at every marker point in the source.

### Keyframe Tools
* **Remap Keys:** Shift the timing of selected keyframes by a specific frame step.

## Installation

1. Copy the extension folder to:
   * **Win:** `C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\`
   * **Mac:** `/Library/Application Support/Adobe/CEP/extensions/`
2. In After Effects, go to `Window > Extensions > Fast Layers`.
3. Enable script writing in `Preferences > Scripting & Expressions > Allow Scripts to Write Files...`.

## 🇷🇺

## Основные функции

### Null Object & Solids
* **Null Object:** Быстрое создание нулл-объектов. Кнопка **Link** привязывает выбранные слои к новому нуллу.
* **Solid Layer:** Создание цветных слоев. Параметры ширины и высоты настраиваются в ⚙️ **Settings**. Если поля пусты — создается по размеру композиции или выбранного слоя. Кнопка **Blind** привязывает выбранные слои к цветному слою.

### Adjustment & Camera
* **Adjustment:** Создание корректирующих слоев. Кнопка **Chain** привязывает выбранные слои к корректирующим слоям.
* **Camera:** Создание камеры.

### Marker Split (New!)
Инструмент для автоматической нарезки слоев по маркерам.
* Поддерживает маркеры композиции и маркеры на отдельных слоях.
* Кнопка **🔄** обновляет список доступных источников.
* Кнопка **Trim** нарезает выбранные слои на части во всех точках, где установлены маркеры.

### Keyframe Tools
* **Remap Keys:** Изменение тайминга выбранных ключей с заданным шагом (в кадрах).

## Установка

1. Скопируйте папку расширения в директорию:
   * **Win:** `C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\`
   * **Mac:** `/Library/Application Support/Adobe/CEP/extensions/`
2. В After Effects перейдите в `Window > Extensions > Fast Layers`.
3. Разрешите скриптам писать файлы и выходить в сеть (`Preferences > Scripting & Expressions > Allow Scripts to Write Files...`).
